# wenote-service
API service for WeNote app

### How to run
Check out the `Makefile`

```bash
// Run api service
$ make run

// Build the app
$ make build:

// Build docker image
$ make docker-build:

// Run docker image on local, must be build explitcitly first
$ make docker-run:

// Stop docker container on local
$ make docker-stop:
```
